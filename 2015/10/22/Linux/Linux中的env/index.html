Linux 中的 env 是什么

env 是环境 Environment 的缩写，正如它的名字一样，它用于 ** 显示系统中已存在的环境变量 **，以及 ** 在定义的环境中执行指令 **，一般位于 /usr/bin/env。

env 作为脚本解释器指定程序

用 env 启动是因为脚本解释器在 linux 可能被安装于不同目录，所以要在 PATH 环境变量中找
