  1. https 访问
    仅为 github.com 设置 socks5 代理 (推荐这种方式，公司内网就不用设代理了，多此一举):
    git config --global http.https://github.com.proxy socks5://127.0.0.1:1086
    其中 1086 是 socks5 的监听端口，这个可以配置的，每个人不同，在 macOS 上一般为 1086.
