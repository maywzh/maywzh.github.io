模块

模块，又称构件，是能够单独命名并独立地完成一定功能的程序语句的集合（即程序代码和数据结构的集合体）

一个独立的模块需要能够独立完成一个功能，可以引用依赖和被依赖。例如 C 语言中的库和头文件，Java 中的包，等等。
